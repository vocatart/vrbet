## Universal Labels

| Label | Usage |
|:-----:|:------|
|  sil  | Silence |
|  pau  | Short Pause |
|  bre  | Breath, Inhale  |
|  asp  | Non-phonemic Aspiration, Exhale |
|  glo  | Non-phonemic Glottal Stop |
|  fry  | Non-phonemic Glottal Fry |
|  que  | (TTS) Question |
|  exc  | (TTS) Exclamation |
